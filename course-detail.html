Computer Programming နဲ့ပတ်သက်ပြီး တစ်လုံးမှ မသိသေးတဲ့သူတွေ / Programming ကိုနားလည်သလိုလိုတော့ရှိတယ် သိပ်မသိဘူးဆိုတဲ့သူတွေအတွက် ဒီသင်တန်းကနေ ကျွမ်းကျင်ပိုင်နိုင်သွားအောင် မိတ်ဆက်ပေးမှာဖြစ်ပါတယ်... JavaScript ကို ES 5 Version နဲ့ အစကနေစပြီး ES 6 အပါအဝင် နောက်ပိုင်း Version တွေဖြစ်တဲ့ ES Next Version တွေနဲ့ Project ရေးနိုင်တဲ့အထိ သင်ကြားပေးသွားမှာဖြစ်ပါတယ်... ဒါလေးရေးလို့ ဒီလိုလေးထွက်လာတယ်လို့ သင်မှာမဟုတ်ပဲ Code တစ်ကြောင်းချင်းစီရဲ့နောက်ကွယ်မှာ ဘယ်လိုအလုပ်လုပ်တယ် ... ဘာကြောင့် ဒီ Code ကိုရေးရတယ်ဆိုတာကအစ သီအိုရီနဲ့ အသေးစိတ် သင်ကြားပေးသွားမှာဖြစ်ပါတယ်...

  • Planning

    Developer တစ်ယောက်အနေနဲ့ Code ရေးတော့မယ်ဆိုရင် သိထားသင့်တာတွေ စဉ်းစားတွေးခေါ်ပုံတွေ Project ရေးဖို့အတွက် ပြင်ဆင်ပုံတွေကို စနစ်တကျ သင်ကြားပေးသွားမှာဖြစ်ပါတယ်... JavaScript ရဲ့ အခြေခံပိုင်းမှာဆိုရင် variables, if/else, operators, boolean logic, functions, arrays, objects, loops, strings, စတာတွေကို အသေးစိတ်သင်ပေးသွားမှာဖြစ်ပြီး ES 6 Version မှာပါဝင်တဲ့ arrow functions, destructuring, spread operator အကြောင်းတွေအထိရယ်... JavaScript ရဲ့ OOP Concept တွေဖြစ်တဲ့ Classes, constructors, prototypal inheritance, encapsulation တို့အကြောင်းတွေကိုရှင်းပြမှာဖြစ်ပါတယ်...

Description

JavaScript is one of the most popular programming languages in the world. Companies like Walmart, Netflix, and PayPal run big internal applications around JavaScript.
